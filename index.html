About App

Real time exchange rate app.

Currency Exchange Rate based on Myanmar.

MMK Exchange Rate App is showing real time currency exchange rate from Central Bank of Myanmar Exchange Rate API Service.

User can use tab icon for other bank eg.CB, AYA..,

To get latest exchange rate information, user will need to pull down the screen, system will also retrieve automatically latest exchange rate information once app is up.
